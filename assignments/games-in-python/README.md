
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a command-line Hangman game in Python that reinforces string manipulation, loops, conditionals, and handling user input. Students will implement game logic and a simple text-based user interface.

## 📝 Tasks

### 🛠️	Core Implementation

#### Description
Implement the core Hangman gameplay: select a secret word, accept letter guesses, reveal correct letters, and track remaining attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list.
- Prompt the player for single-letter guesses and validate input.
- Display the current word progress using underscores for hidden letters (e.g., _ a _ _ m a n).
- Track and display incorrect guesses and remaining attempts.
- End the game when the word is fully guessed or attempts are exhausted, and show a win/lose message.


### 🛠️	Optional Enhancements

#### Description
Add one or more extra features to improve the game experience.

#### Requirements
Completed program should include one or more of the following (optional):

- Load words from an external text file instead of a hardcoded list.
- Allow the player to guess the entire word.
- Display previously guessed letters to the player.
- Add difficulty levels that change the number of allowed attempts.
