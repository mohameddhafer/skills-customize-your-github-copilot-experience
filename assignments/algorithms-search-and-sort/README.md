# 📘 Assignment: Algorithms — Search and Sort

## 🎯 Objective

Implement common search and sorting algorithms in Python, measure their performance on different input distributions, and analyze their time and space complexity.

## 📝 Tasks

### 🛠️ Implement Algorithms

#### Description
Implement the following functions in `starter-code.py`:

- `linear_search(arr, target)` — linear search
- `binary_search(arr, target)` — binary search (for sorted arrays)
- `insertion_sort(arr)`
- `merge_sort(arr)`
- `quick_sort(arr)`

#### Requirements

- Each function must accept appropriate inputs and return correct outputs (see `starter-code.py`).
- `binary_search` should operate on a sorted list and return the index of the target or `None` if not found.
- Sorting functions should return a new sorted list without modifying the original input.

### 🛠️ Experimentation and Analysis

#### Description
Use the provided runner or write a short script to:

- Generate lists of various sizes and distributions (random, sorted, nearly sorted).
- Measure execution time for each algorithm on these datasets.

#### Requirements

- Record timing results and compare observed performance to theoretical complexity.
- Summarize findings in a short report (add `results.txt` to the assignment folder or paste the summary into this README).
